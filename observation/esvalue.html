observation.esvalue

Created on Mon Aug 2 14:51:23 2021

@author: philippe@loco-labs.io

ESValue is a class dedicated to structured data (such as dates, location or measurable properties) and groups common properties and concepts.

ESValue is build around two attributes :

  • 'name' which is a simple String

  • 'value' which corresponds to a more or less complex object :

    • 'DatationValue' : value is a TimeSlot Object which represent a set of time intervals
    • 'LocationValue' : value is a Shapely Geometry which represent a set of polygons
    • 'PropertyValue' : value is a simple dictionary which specifies all the characteristics of a property
    • 'NamedValue' : value can be any simple object
    • 'ExternValue' : value can be any other object

This module groups the classes of the objects used in the observation.esobservation module :

  • DatationValue,
  • LocationValue,
  • PropertyValue,
  • NamedValue
  • ExternValue

and the parent class :

  • observation.esvalue_base.ESValue

  • The concepts of 'ES value' are describe in this page.

419        '''
420        PropertyValue constructor.
421
422        *Parameters*
423
424        - **val** :  property dict or json string (default nullPrp)
425        - **name** :  string (default nullName)
426        - **prp_dict** : boolean(default False) - if True type property has to be in the type dictionary
427        '''
428        ESValue.__init__(self)
429        if isinstance(val, self.__class__):
430            self.name = val.name
431            self.value = val.value
432            return
433        if not val is None and name == ES.prp_type:
434            name = None
435            val = {ES.prp_type: val}
436        elif isinstance(val, str) and isinstance(name, str) and name != ES.nullName:
437            val = {name: val}
438        if isinstance(val, dict):
439            if len(val) > 0 and isinstance(list(val.values())[0], dict):
440                self.name = list(val.keys())[0]
441                self.value |= val[list(val.keys())[0]]
442            else:
443                self.value |= val
444        elif isinstance(val, str):
445            name = val
446        # elif not val is None: raise ESValueError('type data not compatible with PropertyValue')
447        # else: raise ESValueError('type data not compatible with PropertyValue')
448
449        if self.name == ES.nullName and isinstance(name, str) and name != ES.nullName:
450            self.name = name
451        if not ES.prp_type in self.value:
452            raise ESValueError("type property not defined")
453        if not isinstance(self.value[ES.prp_type], list):
454            if prp_dict and not self.value[ES.prp_type] in ES.prop:
455                raise ESValueError(
456                    "property not present in standard dictionnary")
457            if prp_dict:
458                self.value[ES.prp_unit] = ES.prop[self.value[ES.prp_type]][5]

464    def link(self, other):
465        '''
466        return the link (string) between self.value and other.value :
467        - equals     : if self and other are the same
468        - disjoint   : if the self's key/val are all different from other's key/val
469        - within     : if all self's key/val are included in other's key/val
470        - contains   : if all other's key/val are included in self's key/val
471        - intersects : in the others cases'''
472        if self.isEqual(other, name=False):
473            return 'equals'
474        sprp = self._setprp(self.value[ES.prp_type])
475        oprp = self._setprp(other.value[ES.prp_type])
476        if oprp == sprp:
477            union = other.value | self.value
478            union2 = self.value | other.value
479            if union == self.value and union2 == self.value:
480                return 'within'
481            if union == other.value and union2 == other.value:
482                return 'contains'
483            if union == union2:
484                return 'disjoint'
485            return 'intersects'
486        if sprp == sprp | oprp:
487            return 'contains'
488        if oprp == sprp | oprp:
489            return 'within'
490        if oprp & sprp == set():
491            return 'disjoint'
492        else:
493            return 'intersects'
494        return 'undefined'

511        option = {'encoded': False} | kwargs

513        for k, v in self.value.items():
514            if k in [ES.prp_type, ES.prp_unit, ES.prp_sampling, ES.prp_appli, ES.prp_EMFId]:
515                if v != ES.nullDict:
516                    li[k] = v
517            elif k in [ES.prp_period, ES.prp_interval, ES.prp_uncertain]:
518                if v != ES.nullInt:
519                    li[k] = v
520            else:
521                li[k] = v
522        if option['encoded']:
523            return json.dumps(li, ensure_ascii=False, cls=ESValueEncoder)
